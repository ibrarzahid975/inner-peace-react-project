# Inner Peace React Project

This is a React.js project based on the **Tooplate 2143 Inner Peace** template.  
It showcases mindfulness practices, resources, and tools with a clean, responsive interface.

## Features

- Responsive design for desktop and mobile
- Smooth scrolling navigation with active section highlighting
- Mindfulness practices timeline with detailed descriptions
- Resources section with tabs: Meditations, Courses, Tools
- Contact form with alert notification on submit
- Dynamic header background and shadow on scroll

## Tech Stack

- React.js
- HTML5 & CSS3
- JavaScript
